Password Manager

This is an easy secure and reliable password manager.

Here are some simple directions to follow:

  • Step 1: Click the "Download" button above.
  • Step 2: Open the downloaded ZIP file.
  • Step 3: Extract the contents of the ZIP file.
  • Step 4: Open the Password Manager application through vscode.
  • Step 5: Go to main.py and run in terminal
